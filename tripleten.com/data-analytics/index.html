Machine Learning (ML)
8 weeks, 4 projects
Data Science has many facets. One of the most widely-used is Machine Learning (ML)—teaching computers to find the best algorithms to perform specific tasks, creating models that can make predictions and inferences about the real world.Machine Learning builds on everything you've covered so far—stats, Python, and software engineering—to enable you to create intelligent systems. You'll cover supervised learning (including regression and classification models), and learn how to explain Machine Learning and apply it to real-world business projects.
